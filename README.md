![](https://i.imgur.com/UB4ZxM4.jpg)

2. 按下EXIT按鈕離開遊戲

3. 按下SCORE按鈕進入分數排行畫面
4. 在此畫面中，按下BACK按鈕返回遊戲起始畫面，按下EXIT離開遊戲
![](https://i.imgur.com/Xj4oJr3.jpg)

5. 完成所有進球後顯示最後分數
